The instruction can be found at:
https://gildas-lormeau.github.io/zip.js/

the zip file can be downloaded from:
https://github.com/gildas-lormeau/zip.js/zipball/master

As instructed:

1) copy zip.js, z-worker.js, deflate.js, and inflate.js in the folder 'tools'
2) load zip.js in index.html: <script type="text/javascript" src="tools/zip/zip.js"></script>
3) set zip.workerScriptsPath (i.e. the URL to the directory containing z-worker.js, deflate.js and inflate.js)
    i.e.:
    zip.workerScriptsPath = 'tools/zip/'
